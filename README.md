# Describí-Fácil IA

Aplicación web desarrollada con Streamlit e Inteligencia Artificial para ayudar a emprendedores a generar descripciones de productos de manera rápida y profesional.

## Funcionalidades
- Generación automática de descripciones
- Selección de estilo de texto
- Interfaz simple e intuitiva

## Tecnologías utilizadas
- Python
- Streamlit
- OpenAI API
- Visual Studio Code

## Nota
La aplicación funciona en modo demostrativo debido a limitaciones de crédito en la API de OpenAI.
